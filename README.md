# TrixtaLogin
A authentication plugin for Sponge Minecraft Servers.

![](https://i.ibb.co/fHn4rhX/trixtalogin.png)

# How to configure it?

Copy **trixtalogin.conf** in your config sponge folder, and modify "wait_time" as you prefer.
